# Career Agent Documentation Portal

This is the GitHub Pages site for the [career-agent](https://github.com/nextwebb/career-agent) project.

## Deployment

This site is automatically deployed via GitHub Pages from the `docs/` folder on the `main` branch.

Visit: **https://nextwebb.github.io/career-agent/**

## Design

Built with:
- Tailwind CSS (CDN)
- Google Fonts (Geist, Inter, JetBrains Mono)
- Material Symbols
- Custom dark/light theme system

Based on the "Technical Precision" design system - a developer-focused aesthetic with:
- Dark mode default
- High-contrast typography
- Functional color semantics
- Terminal-inspired components
