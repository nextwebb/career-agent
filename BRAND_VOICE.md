## Terminology Standards

### ✅ USE These Terms

| Term | When | Example |
|------|------|---------|
| **Agentic workflow** | Primary identity | "Career Agent is an agentic workflow..." |
| **Skills** | Individual commands | "The `/source` skill finds jobs" |
| **Automation** | What we provide | "Automate ATS form filling" |
| **Claude** | The AI | "Claude generates your CV" |
| **Multi-step** | Process description | "Multi-step AI automation" |
| **AI-powered** | Capability emphasis | "AI-powered job search" |
| **Plugin** | Installation context only | "Install the plugin with..." |

### ❌ AVOID These Terms

| Term | Why | Use Instead |
|------|-----|-------------|
| "Autonomous agent" | Overpromises - human is in the loop | "Agentic workflow" |
| "AI agent" alone | Implies full autonomy | "Agentic workflow" |
| "App" or "Application" | Implies standalone GUI | "Workflow" |
| "Tool" | Too generic/passive | "Agentic workflow" |
| "Platform" | Too enterprise | "Workflow" |
| "Bot" | Feels impersonal | "Claude" or "workflow" |

## Voice & Tone

### Voice Characteristics

**Professional but approachable**
- Technical accuracy without jargon
- Clear explanations
- Assumes intelligence, not expertise

**Example:**
```
✅ "Claude analyzes the job description to emphasize your most relevant experience"
❌ "Our AI uses NLP to parse JD semantics and optimize CV bullet points"
```

**Empowering, not replacive**
- "Automates the tedious parts"
- "Keeps you in control"
- "Handles the work while you focus on strategy"

**Example:**
```
✅ "Claude fills the ATS form. You review and submit."
❌ "Fully automated job applications. Set it and forget it."
```

## FAQ Standard Answers

### "Is it a plugin or an agent?"

```
Career Agent is an agentic workflow delivered as a Claude Code plugin.
It provides AI-powered automation for job applications through a series
of intelligent, multi-step workflows. You install it as a plugin, but
experience it as an intelligent workflow system.
```

### "How is it different from a Chrome extension?"

```
Career Agent is a Claude Code workflow, not a browser extension. It uses
Claude AI to orchestrate the entire application process—from finding roles
to generating tailored CVs to filling forms. The browser automation is just
one part of a larger intelligent workflow.
```

### "Is it fully automated?"

```
No, and that's intentional. Career Agent automates the tedious parts
(searching, CV generation, form filling) but keeps you in control of the
final decisions. Claude never submits applications without your review.
This ensures quality and gives you confidence in every submission.
```
